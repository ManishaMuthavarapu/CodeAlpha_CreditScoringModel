# CodeAlpha Credit Scoring Model

## Overview
This project predicts the creditworthiness of individuals using Machine Learning algorithms.

## Features
- Data preprocessing
- Label encoding
- Random Forest Classifier
- Accuracy evaluation
- Confusion matrix visualization

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset
Credit Score Classification Dataset from Kaggle.

## Model Used
Random Forest Classifier

## Accuracy
Achieved approximately 85% to 90% accuracy depending on dataset preprocessing.

## Conclusion
The machine learning model successfully predicts credit scores based on financial history and customer details. This project demonstrates classification techniques and model evaluation methods.
